One operating system function is the controlling of peripheral devices. Peripheral devices can be anything from printers to drivers, which it does through device management. Device management is where the operating system communicates with the drivers, and allocates resources accordingly. It additionally keeps tracks of these devices. 
Another system function of an operating system is Memory Managment. Memory Managment is is the managment of the Main memory in the form of bytes with individual addresses, accessible by the CPU. The OS allocates memory based on the process that has subsequently requested it. In this way, it decides who and what uses memory.
